How to Convert Images to PDF

  1. Click the upload field and select one or more image files
  2. Hold Ctrl (Windows) or Cmd (Mac) to select multiple images at once
  3. Click "Convert to PDF" — each image becomes one page
  4. Download the resulting PDF file
